## Design Tokens

**Color Scheme:** Deep purple / pink / dark theme

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#7c3aed` | Buttons, links, accents |
| Secondary | `#ec4899` | Highlights, badges |
| Accent | `#a855f7` | Subtle tertiary highlights |
| Background | `#1a0533` | Dark UI backgrounds |
| Surface | `#0f0020` | Card/panel backgrounds |

## Contributing Assets

When adding new assets, please:
- Use SVG format where possible for scalability
- Follow the color scheme above
- Keep file sizes reasonable (optimise PNGs/SVGs)
- Name files descriptively using `kebab-case`

# Feature Settings Implementation

## Gallery View Settings
- Position: Top, Bottom, Left, Right (default: Bottom)
- Thumbnail size: Small, Medium, Large
- Auto-hide delay: 3s, 5s, 10s, Never
- Show page numbers: On/Off

## Smart Search Settings
- Enable natural language: On/Off
- Show live preview: On/Off

## AI Recommendations Settings
- Enable ML recommendations: On/Off
- Algorithm: Hybrid (default), Collaborative only, Content only
- Update frequency: Weekly, Daily, Manual
- Number of recommendations: 5, 10, 20

## Scale Mode Settings
- Remember per manga: On/Off
- Default mode: Fit Screen, Fit Width, etc.

## Implementation
Add to ReaderPreferences.kt and create UI in settings screen.

# Manga Reader App - Project Plan

## Overview
Fork of Komikku with Perfect Viewer reading features + AI recommendations + Gallery view + Dynamic theming

## Features

### Phase 1: Core Improvements
- [ ] Perfect Viewer reading modes (fit width, fit height, original size)
- [ ] Auto-crop margins
- [ ] Better zoom/pan gestures

### Phase 2: AI Features
- [ ] Recommendation system based on reading history
- [ ] Similar manga suggestions

### Phase 3: UI Features
- [ ] Full gallery view (all pages thumbnail grid)
- [ ] Dynamic theming from cover colors

### Phase 4: Polish
- [ ] Performance optimization
- [ ] Bug fixes
- [ ] Release

## Tech Stack
- Kotlin
- Jetpack Compose
- SQLDelight
- Coil (images)
- Android Palette API (theming)

## Branches
- `main` - Stable code
- `feature/reader-modes` - Perfect Viewer features
- `feature/recommendations` - AI recommendations
- `feature/gallery-view` - Gallery view
- `feature/dynamic-theme` - Dynamic theming
